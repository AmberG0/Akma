## SQL файлы для обновления БД:

1. **update_database.sql** — полный скрипт обновления:
   - Добавление поля Status в Orders
   - Создание таблицы Works
   - Проверка структуры

2. **add_status_to_orders.sql** — только добавление статуса

3. **reviews_table.sql** — создание таблицы Reviews с тестовыми данными

## Инструкция по обновлению БД:

```bash
mysql -u root -p construction_site < update_database.sql
```

Или выполните вручную:
```sql
ALTER TABLE Orders 
ADD COLUMN IF NOT EXISTS Status ENUM('новая', 'в работе', 'выполнена', 'отменена') DEFAULT 'новая' AFTER Other_inform;

UPDATE Orders SET Status = 'новая' WHERE Status IS NULL OR Status = '';
```

## Структура статусов:

| Статус | Цвет | Описание |
|--------|------|----------|
| новая | 🟡 Жёлтый | Только что созданная заявка |
| в работе | 🟢 Зелёный | Менеджер назначен, работа идёт |
| выполнена | 🔵 Голубой | Работа завершена |
| отменена | ⚫ Серый | Заявка отменена |

## Тестирование:

1. Создайте тестовую заявку через пользовательскую часть
2. Зайдите в админку (`/pages/admin/orders.php`)
3. Проверьте, что заявка имеет статус "новая" и выделена жёлтым
4. Назначьте менеджера через кнопку "Назначить"
5. Проверьте, что статус изменился на "в работе"

## Примечания:

- Фотографии услуг сохраняются в `/uploads/services/`
- При загрузке проверяется тип файла (JPEG, PNG, GIF, WebP) и размер (макс. 5MB)
- Отзывы публикуются только после модерации в админке
