```bash
mysql -u root -p construction_site < update_status_fixed.sql
```

## 🚀 Инструкция по применению:

1. **Выполнить SQL для статусов:**
```bash
mysql -u root -p construction_site < update_status_fixed.sql
```

2. **Проверить права на папку uploads:**
```bash
chmod 777 /workspace/photoservice/uploads/services
```

3. **Протестировать загрузку фото:**
   - Зайти в админку → Услуги → Добавить услугу
   - Выбрать файл изображения
   - Сохранить - фото должно сохраниться в uploads/services/

4. **Протестировать изменение статуса:**
   - Зайти в админку → Заявки → Просмотр заявки
   - Изменить статус в форме
   - Нажать "Изменить статус"

## 🔧 Технические детали:

- Загрузка фото: проверка типа (JPEG, PNG, GIF, WebP) и размера (макс. 5MB)
- Генерация уникального имени файла: `service_TIMESTAMP_UNIQID.ext`
- Валидация карты: номер (13-19 цифр), срок (ММ/ГГ), CVV (3 цифры), имя (латиница)
- Статусы: ENUM('новая', 'в работе', 'выполнена', 'отменена')
