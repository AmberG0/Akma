ИНСТРУКЦИЯ ПО УСТАНОВКЕ И ОБНОВЛЕНИЮ БАЗЫ ДАННЫХ
================================================================================

1. СОЗДАНИЕ ТАБЛИЦЫ ОТЗЫВОВ (Reviews)
-------------------------------------------
Выполните файл reviews_table.sql:

mysql -u root -p construction_site < /workspace/photoservice/reviews_table.sql

Или скопируйте содержимое файла reviews_table.sql и выполните в phpMyAdmin.


2. ДОБАВЛЕНИЕ СТАТУСОВ ЗАЯВОК (Status)
-------------------------------------------
Если при добавлении поля Status возникла ошибка, выполните исправленный SQL:

mysql -u root -p construction_site < /workspace/photoservice/update_status_fixed.sql

Или скопируйте содержимое файла update_status_fixed.sql и выполните в phpMyAdmin.


3. ПРОВЕРКА СТРУКТУРЫ ТАБЛИЦЫ ORDERS
-------------------------------------------
DESCRIBE Orders;

Должно отображаться поле Status с типом ENUM('новая', 'в работе', 'выполнена', 'отменена')


4. ПАПКА ДЛЯ ЗАГРУЗКИ ФОТО
-------------------------------------------
Папка уже создана: /workspace/photoservice/uploads/services/
Права доступа: 777 (чтение/запись для всех)

Если фото не загружаются, проверьте права:
chmod -R 777 /workspace/photoservice/uploads/


5. ТЕСТИРОВАНИЕ
-------------------------------------------
а) Создание услуги с фото:
   - Зайдите в админку: /photoservice/pages/admin/login.php
   - Логин: admin, Пароль: password
   - Перейдите в "Услуги" -> "Добавить услугу"
   - Заполните форму и загрузите изображение
   - Проверьте, что фото появилось в папке uploads/services/

б) Оформление заявки с оплатой картой:
   - Добавьте услугу в корзину
   - Перейдите в "Подборка услуг"
   - Заполните форму, включая данные карты
   - Данные карты проходят валидацию:
     * Номер карты: 13-19 цифр
     * Срок действия: ММ/ГГ (не истёкший)
     * CVV: 3 цифры
     * Имя держателя: латинские буквы
   - Отметьте согласие на обработку персональных данных
   - Оформите заявку

в) Проверка статусов заявок:
   - Зайдите в админку -> "Заявки"
   - Новые заявки выделены жёлтым цветом с пульсацией
   - Заявки без менеджера выделены красным
   - Можно изменить статус: новая, в работе, выполнена, отменена


6. ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ
-------------------------------------------
Проблема: Ошибка #1064 при добавлении поля Status
Решение: Использовать update_status_fixed.sql вместо старого запроса

Проблема: Фото не загружаются
Решение: 
   - Проверить права на папку uploads/services/
   - Проверить размер файла (макс. 5MB)
   - Проверить формат файла (JPEG, PNG, GIF, WebP)

Проблема: Не отображаются заявки в админке
Решение:
   - Проверить наличие поля Status в таблице Orders
   - Выполнить update_status_fixed.sql
   - Проверить подключение к БД в i/WebsiteBackend/db.php

Проблема: Не работает валидация карты
Решение:
   - Проверить JavaScript консоль на ошибки
   - Убедиться, что все поля заполнены корректно
   - Проверить формат срока действия (ММ/ГГ)
