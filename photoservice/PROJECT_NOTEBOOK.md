6. СТРУКТУРА БАЗЫ ДАННЫХ (construction_site)
================================================================================
Таблицы:
1. Category - категории услуг
   - ID_category (PK)
   - Name
   - Description

2. Services - услуги
   - ID_services (PK)
   - Name
   - Description
   - Unit (единица измерения)
   - Price
   - Category (FK -> Category.ID_category)
   - Relevance (ENUM: 'да', 'Нет')
   - Photo (путь к изображению)

3. Personnel - персонал/модераторы
   - ID_personal (PK)
   - Login (UNIQUE)
   - Password
   - Fio
   - Role
   - Num_phone
   - Mail

4. Services_tab - связка заявок и услуг
   - Service_tab_ID (PK)
   - Description
   - Data_create
   - Data_start
   - Service_list_ID (FK -> Services.ID_services)
   - Service_list_count
   - Count_pay

5. Orders - заявки
   - ID_order (PK)
   - Client
   - Num_phone
   - Mail
   - Service_tab_ID (FK -> Services_tab.Service_tab_ID)
   - Time_the_bell
   - Type_pay
   - Face_client
   - Accept_order_Per_ID (FK -> Personnel.ID_personal)
   - Other_inform
